Разработка и продвижение сайта электротехнической экспертизы

Главной задачей проекта являлось создание продающего сайта и привлечение зявок на электротехническую экспертизу. Бесплатное ведение рекламных компаний по модели CPA (оплата только за привлеченных клиентов). Адаптация сайта под запросы рекламных компаний, чтобы потенциальные клиенты попадали на страницы с информацией, которую они искали

Функционал и особенности веб-приложения:

  • Адаптивный дизайн. 37 отдельных страниц
  • Ориентированная на пользователя форма заявки (посетитель может быстро, без ввода капчи заполнить и отправить заявку на услуги компании)
  • Настраиваемый спам фильтр(возможность тонкой настройки по различным критериям)
  • Всплывающие уведомления в случае ошибки, успешной отправки заявки или обнаружения спама
  • Настройка по чтового сервера, рассылка автоматического уведомления клиенту на почту в случае оправки заявки

Применяемые технологии:

  • Язык программирования Python 3.10
  • Веб-фреймворк Flask
  • Шаблонизатор Jinja2
  • Набор инструментов Werkzeug
  • Фреймворк Bootstrap 5
  • CSS, HTML

Результаты:

  • Окупаемость вложений через 2 месяца
  • Среднее количество целевых заявок в месяц: 11
  • Средняя стоимость привлечения целевой заявки: 550 р. с НДС
